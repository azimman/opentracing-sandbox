# Getting things to work

## Requirements

```
$ pip install django_opentracing Flask-Opentracing lightstep
```

## Running things

For flask:

```
python/flask_fib$ python flask_fib.py
```

For django:

```
python/django_fib$ python manage.py runserver 8000
```

## In your web browser

`http://localhost:8000/client/fib`
